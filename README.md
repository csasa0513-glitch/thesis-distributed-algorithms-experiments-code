# Experiments for Chapter 6

This folder contains the Python code that produces every figure and table
in Chapter 6 of the thesis "Distributed Algorithms on Complex Graphs".

## Layout

```
experiments/
├── README.md                # this file
├── requirements.txt         # pinned dependencies
├── config.py                # all experimental parameters (N, L, seeds, ...)
├── games/
│   └── nash_cournot.py      # Cournot instance, F_i, projections, offline NE
├── graphs/
│   └── topologies.py        # networkx generators + Koshal-style W
├── algorithms/
│   ├── sync_koshal.py       # Algorithm 1  (synchronous)
│   └── async_gossip.py      # Algorithm 2  (asynchronous gossip)
├── analysis/
│   ├── metrics.py           # lambda_2, clustering, diameter, degree stats
│   └── plots.py             # matplotlib figures used in the thesis
├── run_sync.py              # Q1: synchronous algorithm
├── run_async_regular.py     # Q2: asynchronous on 4 topologies
├── run_async_sensitivity.py # Q3: sensitivity sweep of WS(p)
└── results/                 # csv + png outputs (regenerated by the scripts)
```

## How to reproduce every figure and table

Install dependencies once:

```bash
pip install -r requirements.txt
```

Then run the entry scripts in order:

```bash
# Sec. 6.1 -- spectral gap |lambda_2(W)| (Table 1)
python run_spectral.py

# Sec. 6.2 -- synchronous algorithm on 4 topologies
python run_sync.py

# Sec. 6.3 -- asynchronous algorithm on 4 topologies
python run_async_regular.py

# Sec. 6.4 (async) and 6.5 (sync) -- sensitivity sweep of WS(p)
python run_async_sensitivity.py
python run_sync_sensitivity.py
```

Total runtime is roughly 10-15 hours on a standard laptop.

All experimental parameters (the seed, the number of sample paths,
the reporting horizons, the convergence tolerance) are collected in
`config.py`, so they can be changed in a single place.

## Reproducibility

* All random draws are seeded from `config.SEED` (default `20260327`).
* The Nash--Cournot coefficients are drawn once in `config.py` and kept
  fixed across topologies; `config.resample(N)` redraws them for a
  different number of players with seed `SEED + N`.
* Both `run_sync.py` and `run_async_regular.py` use `R = 50` sample
  paths per cell, matching Koshal et al. (2016). The constant-stepsize
  variant of Koshal (2016, §4.3) is omitted; only the diminishing
  stepsize is used.
* Output CSVs fill Tables 5--7 and the sensitivity table in Chapter 6.
